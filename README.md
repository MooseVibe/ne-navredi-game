# НЕ НАВРЕДИ — прототип смен

Внутренний игровой прототип IQ-Provision для демонстрации коллегам. Сценарии и продуктовые утверждения требуют медицинской и юридической проверки перед публичным использованием.

## Локальный запуск

```bash
python3 -m http.server 8000
```

Открой `http://localhost:8000`.

## Деплой и возврат

- GitHub Pages публикует корень ветки `main`;
- предыдущая публичная версия сохранена в коммите `44636cc`.

Для возврата восстанови этот коммит в `main` — Pages развернёт его снова.
