# Не навреди

Браузерная игра-симулятор ночной смены в аптеке.

## Запуск локально

Откройте `index.html` в браузере или запустите локальный сервер:

```sh
python3 -m http.server 8000
```

Затем откройте <http://localhost:8000>.

## Публикация

Проект не требует сборки. Для GitHub Pages: **Settings → Pages → Deploy from a branch → `main` / `/ (root)`**.
